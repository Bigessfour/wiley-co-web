# Blazor Documentation Index

This workspace keeps the hard-copy Blazor reference material in the local `Blazor Documentation/` folder for Copilot and maintainer use.

## Usage

- Treat these PDFs as the local reference source for Blazor and Syncfusion questions.
- Prefer local documentation over web lookup when the answer is available in these files.
- Use [docs/blazor-documentation-topics.md](blazor-documentation-topics.md) as the quick topic map before opening the PDFs.
- If a specific topic needs to be searched frequently, add an extracted Markdown or text summary next to this index so the workspace becomes more searchable.
